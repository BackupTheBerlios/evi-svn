You have started EVI for the first time...

EVI is a Java application framework. Its functionalities completely depends on which modules are loaded. (More about EVI)

We plan to support some common internet communication protocols, firstly IRC and hopefully ICQ.
Additionally to creating good client programs for IRC and ICQ we want to provide a number of entertainment modules like games and other tools like a timetable and so on.
See About -> About for more information.

Probably you might want to start now... You might want to do the following things:

  • You might setup the EVI framework: File -> Configuration
  • If your EVI distribution did not come with modules, you probably might want to download some: File -> Modules
  • After all, you might want to automatically start modules when the client is started: File -> Autostart
