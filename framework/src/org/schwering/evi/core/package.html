Contains the module interfaces and the module engine.

There are two major interfaces which must be implemented:

  • IModuleInfo defines meta information
  • IModule must be implemented by the concrete module
Note that generally the class that implements IModuleInfo should not be the same that implements IModule.
The class that implements IModuelInfo must define a constructor that takes no arguments.

There's a number of different interfaces which can be used to give the module certain features and properties.
