    • ICQ communication support (I hope) (exeqlib 0.1?)
  • Entertainment
    • Pong single/multiplayer
    • TicTacToe single/multiplayer
    • possibly Skat multiplayer
  • Tools
    • IRC just in time log streamer to someone else
    • Timetable
    • Calendar
    • SMTP mailbox checker (java.mail)
    • File manager (File)
    • Image viewer (ImageIO, ImageIcon, BufferedImage, Graphics)
    • Simple webbrowser (fast fertig, erweiterbar, JEditorPane)
    • Audio player would be great (MP3 library)
