This directory contains the JOAL and JOGL binaries.

JOGL, GlueGen and JOAL are copyright by their respective owners.

You can find JOGL at http://jogl.dev.java.net
JOGL is a Java OpenGL library.

You can find JOAL at http://joal.dev.java.net
JOAL is a Java Sound3D library.

You can find GlueGen at http://gluegen.dev.java.net
GlueGen is a tool which automatically generates the Java and JNI 
code necessary to call C libraries.
